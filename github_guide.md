# دليل إنشاء حساب GitHub ورفع المشروع

بما أنه ليس لديك حساب GitHub حالياً، إليك الخطوات البسيطة للبدء ورفع مشروعك:

## الخطوة 1: إنشاء الحساب
1. توجه إلى [GitHub.com](https://github.com/).
2. اضغط على **Sign up**.
3. اتبع التعليمات (إدخال البريد الإلكتروني، كلمة المرور، واسم المستخدم).
4. قم بتمكين حسابك من خلال الرسالة التي ستصلك على البريد الإلكتروني.

## الخطوة 2: تهيئة Git على جهازك (مهم جداً)
بمجرد إنشاء الحساب، افتح الـ Terminal في مشروعك وقم بتنفيذ الأوامر التالية لتعريف نفسك لجهازك (استخدم نفس بيانات حسابك الجديد):
```bash
git config --global user.email "بريدك_الإلكتروني@example.com"
git config --global user.name "اسم_المستخدم_الخاص_بك"
```

## الخطوة 3: إنشاء مستودع (Repository) جديد
1. في GitHub، اضغط على علامة **+** في الزاوية العلوية ثم اختر **New repository**.
2. أعطِ المشروع اسماً (مثلاً: `library_management`).
3. اجعله **Public** أو **Private** (حسب رغبتك).
4. **لا تقم** بإضافة README أو .gitignore (لأننا جهزناها بالفعل).
5. اضغط على **Create repository**.

## الخطوة 4: ربط المشروع ورفعه
ستظهر لك صفحة بها أوامر، قم بنسخ الأوامر التي تحت عنوان "push an existing repository from the command line" وتشغيلها في Terminal مشروعك. ستكون غالباً بهذا الشكل:
```bash
git remote add origin https://github.com/اسم_المستخدم/اسم_المستودع.git
git branch -M main
git push -u origin main
```
